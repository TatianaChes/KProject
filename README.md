# Обзор
---
Мобильное приложение, позволяющее повысить производительность труда сотрудника склада, путем облегчения процесса идентификации сырья и сбора технологической информации о нем.

__Функции:__

− сканирование QR-кодов;

− генерирование QR-кодов;

− сохранение QR-кодов;

− запись в базу данных.
